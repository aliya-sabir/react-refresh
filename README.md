# React Refresh

A small React job board built with Vite, React Router, Tailwind CSS, and json-server.

## Run the project

Install dependencies:

```bash
npm install
```

Start the mock API:

```bash
npm run server
```

Start the app in development:

```bash
npm run dev
```

Optional production preview:

```bash
npm run build
npm run preview
```

## Features

- Browse job listings
- View job details
- Add a new job
- Edit an existing job
- Delete a job
